---
Ce dossier contient des jeux de données exemples pour les exercices de la formation.

## Fichiers disponibles

### `esane_secteurs_2022_exemple.csv`
Données structurelles annuelles par section NAF pour la France (2022).
Variables : section_naf, libelle, nb_entreprises, ca_milliards_ht, va_milliards, emploi_milliers_etp, fbcf_milliards

⚠️ **Données illustratives** basées sur les ordres de grandeur ESANE INSEE.
Pour les données officielles : https://www.insee.fr/fr/statistiques/2045265

### `creations_defaillances_mensuelles_exemple.csv`
Séries mensuelles de créations et défaillances d'entreprises en France (2021-2023).
Variables : annee, mois, creations_totales, dont_micro_entreprises, dont_classiques, defaillances

⚠️ **Données illustratives** basées sur les publications INSEE.
Pour les données officielles : https://www.insee.fr/fr/statistiques/2015937

### `sbs_eurostat_exemple.csv`
Statistiques structurelles d'entreprises (SBS Eurostat) pour plusieurs pays et secteurs.
Variables : pays, iso2, nace, libelle_nace, annee, nb_entreprises, ca_milliards, va_milliards, personnes_occupees_milliers

⚠️ **Données illustratives** basées sur les ordres de grandeur Eurostat SBS.
Pour les données officielles : https://ec.europa.eu/eurostat/databrowser/view/sbs_na_ind_r2
